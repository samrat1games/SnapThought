## 🎮 Как использовать

### Создать группу
1. Перейти в меню → Groups
2. Нажать "Create Group"
3. Заполнить данные: название, описание, правила
4. Другие пользователи смогут присоединиться

### Начать трансляцию
1. Перейти в меню → Live
2. Нажать "Go Live"
3. Заполнить название и описание
4. Трансляция станет видна другим пользователям
5. Они могут отправлять сообщения в чат и подарки

### Посмотреть награды
1. Перейти в меню → Awards
2. Вкладка "My Awards" - ваши награды
3. Вкладка "Progress" - прогресс к достижениям
4. Вкладка "Leaderboard" - лучшие пользователи
5. Вкладка "All Awards" - все доступные награды
