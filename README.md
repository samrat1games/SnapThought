# SnapThought

A social media platform — hybrid of Instagram and X (Twitter). Built with vanilla HTML/CSS/JS + Supabase.

![SnapThought](icons/App-logo.png)

---

## English

**SnapThought** is a modern social media web application with features from both Instagram and X.

### Features
- Auth: Email signup + Google OAuth
- Posts with text, images, videos, and image carousels
- Like, repost, comment, bookmark
- User profiles with verified/dev/mod badges
- Follow/unfollow system
- Search (users + posts) with saved searches
- Notifications
- Bookmarks with folders
- Dark/Light theme
- Mobile responsive (Instagram-style bottom nav)
- Admin panel with role management + audit log
- Question/support system
- 7-day username change limit
- Hashtags and @mentions (clickable links)
- Post editing (within 15 min of creation)
- Post sharing (Web Share API + copy link)
- Post views tracking
- User lists (curated feeds)
- Polls (create polls with options in posts)
- Block and mute users
- Post drafts (auto-save)
- Geolocation on posts
- Keyboard shortcuts (N=new, /=search, J/K=navigate)
- Data export (JSON + CSV)
- Trends page (trending hashtags + popular posts)
- ShortV (TikTok-like short videos with camera)
- Discussions (Reddit-like communities)
- Direct Messages (real-time polling)
- Stories (24-hour disappearing posts with viewer)
- Threaded replies (reply to comments)
- Comment likes and editing
- Image carousel (multiple images per post)
- Pinned posts on profile
- Achievement badges (13 types, auto-awarded)
- Referral system (invite friends)
- Rate limiting (client-side)
- Spam detection
- Offline support (IndexedDB caching)

### Tech Stack
- Vanilla HTML/CSS/JS (no frameworks)
- Supabase (auth, database, storage, realtime)
- ES Modules via CDN

### Setup
1. Create a [Supabase](https://supabase.com) project
2. Run `schema.sql` in SQL Editor
3. Edit `js/config.js` with your Supabase URL and anon key
4. Run `npx serve .` and open `http://localhost:3000`
