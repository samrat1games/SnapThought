## 🗄️ Новые таблицы БД (13 таблиц)

### Группы (3 таблицы)
- `groups` - основные данные
- `group_members` - участники
- `group_posts` - посты в группе

### Live Streaming (4 таблицы)
- `live_streams` - трансляции
- `live_stream_viewers` - зрители
- `live_stream_messages` - сообщения чата
- `live_stream_gifts` - подарки

### Награды (3 таблицы)
- `awards` - определение наград
- `user_awards` - награды пользователя
- `award_criteria` - критерии получения

### Вспомогательные (3 таблицы)
- Индексы для оптимизации
- Триггеры для автоинкремента
- Каскадные удаления

## 🔗 Маршруты

```
/groups          → Список всех групп

/live            → Активные трансляции
/live/:streamId  → Просмотр трансляции
/awards          → Центр достижений
```
